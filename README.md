# Contacts
Simple Command Line Application to Manage Contacts

## Requirements
```
jdk 1.8
maven 3.x
```

## Running the application
```
$ git clone https://github.com/yashk/Contacts.git
$ cd Contacts
$ mvn test -Pmain -q
1) Add contact 2) Search 3) Exit
1
Enter name:yash
1) Add contact 2) Search 3) Exit
2
Enter name:ya
yash
1) Add contact 2) Search 3) Exit
3
Happy Searching
```
